## 中医治未病智能辅助技能（Skill）项目介绍

### 一、项目背景

中医“治未病”理念源远流长，涵盖“未病先防、既病防变、瘥后防复”三大核心思想。随着健康中国战略推进及大众健康管理需求提升，如何将传统中医智慧与现代人工智能技术结合，为用户提供个性化、可操作的治未病指导，成为本项目的出发点。

本技能（Skill）旨在作为智能助手（如大语言模型）的专业插件，通过内置中医体质辨识、节气养生、饮食调摄、经络穴位推荐等知识库，辅助用户在日常场景中获得科学、安全的中医预防保健建议。

### 二、项目目标

1. **普及中医治未病理念**：通过自然对话，帮助用户理解“防大于治”的中医健康观。
2. **提供个性化养生指导**：基于用户自述的体质特征、季节、症状等，输出定制化的饮食、运动、情志调节方案。
3. **辅助早期风险识别**：针对常见亚健康状态（如失眠、疲劳、消化不良）给出中医证型倾向提示，并建议就医方向。
4. **安全合规**：所有建议均明确为保健参考，不替代诊断，不推荐具体药品或治疗。

### 三、核心功能模块

| 模块 | 功能说明 |
|------|----------|
| **体质辨识助手** | 通过简短问答（如9种体质量表简化版）判断用户倾向体质（平和、气虚、阳虚、阴虚、痰湿、湿热、血瘀、气郁、特禀），并给出对应调养原则。 |
| **四季/节气养生** | 依据当前节气（如立春、夏至、霜降）及地域气候，推荐当令起居、食疗、导引（如八段锦）要点。 |
| **症状-膳食映射** | 用户输入“最近口干舌燥”“容易腹泻”等，返回适宜的中医食疗方（如“百合麦冬茶”“山药粥”）及禁忌食物。 |
| **经络穴位自护** | 针对常见不适（头痛、肩颈酸、胃胀），指导简易穴位定位与按摩手法（如按揉足三里、太阳穴），并标注注意事项。 |
| **情志调摄建议** | 结合中医五志（怒喜思悲恐）与五行生克，提供情绪调节话术、音乐或呼吸法建议。 |

### 四、技术实现要点

- **知识库构建**：基于《中医体质分类与判定》（ZYYXH/T157-2009）、《黄帝内经》养生理论、权威中医食疗及针灸教材，结构化为规则库与问答对。
- **对话式交互**：支持多轮澄清，例如询问“您的舌苔是否偏白腻？”“手脚心是否容易发热？”以提高辨证准确度。
- **安全约束层**：内置黑名单规则（如禁止对急性重症、孕妇给出未核实建议；明确建议就医的红旗警示症状）。
- **个性化记忆**（可选）：在用户授权下，短期记忆历史建议反馈，避免重复推荐。

### 五、应用场景

1. **个人健康助手**：普通用户日常询问“春天容易犯困怎么办？”“经常长口腔溃疡吃什么？”
2. **中医诊所/养生机构**：辅助健康管理师为客户生成初步调养方案，提升服务效率。
3. **企业健康管理**：集成到员工健康平台，提供节气工间操、办公室茶饮方案等。
4. **教育科普**：帮助中医爱好者理解辨证施膳基础，作为教学互动工具。

### 六、项目价值

- **社会价值**：降低医疗资源占用，推动从“治病”向“防病”转变，符合《“健康中国2030”规划纲要》。
- **用户价值**：24小时可及的中医常识支持，减少盲目进补或错误养生行为。
- **专业赋能**：非中医背景的AI助手可借助本Skill输出有一定深度的建议，同时保持风险可控。

### 七、后续规划

- 加入舌诊/面诊图像描述接口（用户上传文字描述或合规图像分析）。
- 对接权威中药方剂数据库，提供“证-方”对应参考（仅限历史经典方名称及组成，不替代医师开方）。
- 开发效果反馈机制，持续优化推荐逻辑。

### 八、风险与合规声明

本Skill所有内容仅供健康参考，不构成医疗诊断或治疗建议。用户如存在明确疾病或症状持续不缓解，应立即就医。本Skill不提供任何药物购买链接或处方药推荐，不违反互联网诊疗相关法规。
